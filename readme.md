    - [ ] Асимптотический анализ/ Big(O)
    - [ ] Структуры данных:
        - [ ] Массивы
        - [ ] Связные списки
        - [ ] Стек
        - [ ] Очередь
        - [ ] Хэш-таблицы
    - [ ] Дополнительно:
        - [ ] Двоичный поиск
        - [ ] Битовые операции
    - [ ] Деревья:
        - [ ] Деревья

    - [ ] Сортировка:
        - [ ] выбором
        - [ ] вставками
        - [ ] пирамидальная
        - [ ] быстрая
        - [ ] слиянием
    - [ ] Графы:
        - [ ] ориентированные

        - [ ] матрицы смежности

        - [ ] обходы: BFS, DFS
